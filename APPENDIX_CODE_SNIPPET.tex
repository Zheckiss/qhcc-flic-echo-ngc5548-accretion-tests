\section{Code and reproducibility package}

The NGC~5548 FLiC accretion-lag analysis code, runbook, and reproduction scripts are provided as a public GitHub package:
\[
\texttt{https://github.com/Zheckiss/qhcc-flic-echo-ngc5548-accretion-tests}
\]
The repository contains the v4 baseline runner, the v4.4 FLiC lag-layer diagnostic, the v4.5 disk-shape diagnostic, and a compact README with the exact command sequence used to reproduce the Note~II figures.  Raw public reverberation-mapping data are not stored in the repository; they are downloaded or regenerated locally by the scripts.
